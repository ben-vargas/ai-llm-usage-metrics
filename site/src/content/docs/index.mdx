# llm-usage-metrics Documentation

Welcome to the documentation for `llm-usage-metrics` - a CLI tool for aggregating local LLM usage metrics from pi, codex, and opencode sessions.

## Overview

This tool automatically discovers and parses usage data from supported sources, normalizes it into a common format, applies pricing models, and renders aggregated reports in your preferred format.
