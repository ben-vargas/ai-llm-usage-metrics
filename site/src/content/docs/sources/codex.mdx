## Default location

- `~/.codex/sessions/**/*.jsonl`

## Override

```bash
llm-usage daily --codex-dir /path/to/codex/sessions
# or
llm-usage daily --source-dir codex=/path/to/codex/sessions
```

## Parsing notes

The adapter handles Codex token-count events and computes deterministic usage deltas for cumulative counters.
