# codex Source

OpenAI Codex CLI session data adapter.

## Data Location

Codex stores session data in:

- **macOS**: `~/Library/Application Support/codex/`
- **Linux**: `~/.config/codex/`
- **Windows**: `%APPDATA%/codex/`

## Parsed Events

The codex adapter extracts:

- Session metadata
- Model usage and completions
- Token consumption
- Cost estimates based on OpenAI pricing
