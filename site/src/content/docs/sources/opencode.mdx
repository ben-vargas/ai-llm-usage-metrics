# opencode Source

Opencode session data adapter.

## Data Location

Opencode stores session data in:

- **macOS**: `~/Library/Application Support/opencode/`
- **Linux**: `~/.config/opencode/`
- **Windows**: `%APPDATA%/opencode/`

## Parsed Events

The opencode adapter extracts:

- Session metadata
- Model interactions
- Token usage
- Cost calculations
