## Global Options

| Option | Description |
|--------|-------------|
| `--format` | Output format: `terminal`, `json`, `markdown` |
| `--source` | Filter by source: `pi`, `codex`, `opencode` |
| `--help` | Display help information |
| `--version` | Display version information |

## Commands

### Default (no command)

Runs the aggregation and outputs formatted metrics.

```bash
llm-usage [options]
```
