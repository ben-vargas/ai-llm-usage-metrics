# Output Formats

:::caution
This page is a placeholder. Output format documentation will be expanded.
:::

## Terminal (Default)

Rich formatted output with colors and tables optimized for terminal display.

```bash
llm-usage
```

## JSON

Machine-readable JSON output for piping to other tools.

```bash
llm-usage --format json
```

## Markdown

Formatted markdown tables suitable for documentation or GitHub issues.

```bash
llm-usage --format markdown
```
