# Configuration

:::caution
This page is a placeholder. Configuration options will be documented here.
:::

## Configuration File

The CLI can read configuration from:

- `.llm-usagerc` in the current directory
- `~/.llm-usagerc` in the home directory

## Options

Configuration options include:

- Default output format
- Custom data directories
- Pricing overrides
- Source-specific settings
