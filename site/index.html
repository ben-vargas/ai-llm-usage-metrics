Auto-discovery

Point it at your home directory. It finds .pi sessions, .codex logs, and OpenCode databases automatically. Zero config.

Real pricing

Costs pulled from LiteLLM's pricing database and cached locally. Use --pricing-offline when you're on a plane.

Multiple formats

Terminal tables for your eyes, --json for your scripts, --markdown for your docs. Filter by source, provider, or model.

Quick start

Three commands. Daily, weekly, monthly. Add flags to taste.

# today's usage
llm-usage daily

# this week, in Paris time
llm-usage weekly --timezone Europe/Paris

# January 2026
llm-usage monthly --since 2026-01-01 --until 2026-01-31
# only pi and codex sources
llm-usage monthly --source pi,codex

# only Claude models
llm-usage monthly --model claude

# OpenAI provider across all sources
llm-usage monthly --provider openai

# combine filters
llm-usage monthly --source codex --model gpt-4.1
# JSON for scripting
llm-usage daily --json

# Markdown for docs
llm-usage daily --markdown

# per-model breakdown in columns
llm-usage monthly --per-model-columns
# custom session directories
llm-usage daily --pi-dir /path/to/pi --codex-dir /path/to/codex

# explicit OpenCode database
llm-usage daily --opencode-db /path/to/opencode.db

# offline pricing (cached only)
llm-usage monthly --pricing-offline

# or run without installing
npx --yes llm-usage-metrics daily
